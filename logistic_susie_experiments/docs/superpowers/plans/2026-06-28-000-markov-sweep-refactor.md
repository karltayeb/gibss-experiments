# 000_markov sweep-spec refactor — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax.

**Goal:** Replace the 40 hand-named designs/enrichments + over/template collection wiring with a compact `sweep:` collection block that declares the grid once; effects come from a frozen lookup table via a `logbf_single_effect` sampler that receives the design type + b0 injected by the loader.

**Architecture:** A frozen effect-size table `FROZEN_B[design][b0][target_logbf]` lives in `logbf_sizing` (generated offline from `solve_b`, stored as a literal). A new `logbf_single_effect(X, rng, *, design, b0, target_logbf)` sampler reads it. The loader gains a `sweep:` collection block: it takes one `design:` (function + n/p [+freq]), and `rho`/`b0`/`target_logbf` lists, and expands to `{design coord} × {enrichment coord}`, injecting `design` (sizing key) + `b0` into each enrichment's arguments and adding one null per b0. `000_markov.yaml` is rewritten to four `sweep:` blocks; the 24 named designs + 16 enrichment families are removed from `library.yaml`. `viz_dims` reads `logbf` directly from the enrichment's `target_logbf` argument.

**Tech Stack:** Python, numpy, scipy, pyyaml, polars, pytest. `uv run` for all commands.

## Global Constraints

- Use `uv run` for every python/pytest command.
- Dispatch model unchanged: designs/enrichments/methods are `{function, arguments}` resolved by `loader.resolve_callable`. The sweep block only *generates* these coordinates; downstream (MANIFEST, fit, viz_dims, plots) is unchanged.

```python
def logbf_single_effect(X, rng, *, design, b0, target_logbf):
    """One causal feature sized to a target univariate logBF via the frozen
    exact-LRT table (lookup, not a solve). target_logbf==0 => null (no effect).
    `design` is the sizing key, `b0` the intercept — both injected by the loader's
    sweep expansion from the paired design."""
    if target_logbf == 0:
        return [], []
    from simulations.effect.logbf_sizing import FROZEN_B
    b = FROZEN_B[design][float(b0)][int(target_logbf)]
    index = int(rng.integers(0, X.shape[1]))
    return [index], [float(b)]
```

```python
def _expand_sweep(library, sc_name, block):
    import itertools as _it
    from simulations.effect import logbf_sizing
    sw = block["sweep"]
    dspec = dict(sw["design"])
    fn = dspec.pop("function")
    freq = dspec.get("freq")
    design_key = logbf_sizing.DESIGN_KEY[(fn, freq if fn == "binary_markov_X" else None)]
    rhos = list(sw["rho"]); b0s = list(sw["b0"]); lbfs = list(sw["target_logbf"])
    results = []
    for rho in rhos:
        design_coord = {"function": fn, "arguments": {**dspec, "rho": rho}}
        sims, coords = [], []
        # signal cells, then one null per b0
        cells = [(b0, L) for b0 in b0s for L in lbfs] + [(b0, 0) for b0 in b0s]
        for b0, L in cells:
            enr = {"function": "logbf_single_effect",
                   "arguments": {"design": design_key, "b0": float(b0), "target_logbf": int(L)},
                   "intercept": float(b0)}
            coord = {"design": design_coord, "enrichment": enr}
            name = f"{design_key}__b0={b0:g}__" + (f"lbf={L:g}" if L else "null")
            spec = loader_spec_from_coord(coord, f"{sc_name}__{name}")
            sims.append(spec)
            coords.append({"coordinate": coord, "name": spec.name})
        results.append({
            "name": f"{sc_name}__{block.get('name', design_key)}__rho={rho:g}",
            "alias": f"{block.get('name', design_key)} rho={rho:g}",
            "simulations": sims, "coordinates": coords,
        })
    return results
```
